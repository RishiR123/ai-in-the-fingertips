## What You Will Learn

**Part 1** — What makes an AI agent different from a chatbot, the agent loop, tool use, real agent products in the wild, and the risks of autonomous AI.

**Part 2** — Build a minimal AI agent from scratch using Python and the Anthropic API. The agent can think, decide which tool to use, call it, and loop until the task is done.
