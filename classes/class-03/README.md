## What You Will Learn

A deep-dive into Anthropic — the company, the Claude model family, and the products built around it. This session covers why Anthropic was founded, how their approach differs from OpenAI's, and what their ecosystem looks like today — from Claude.ai to the API to their massive cloud partnerships with Amazon and Google.
