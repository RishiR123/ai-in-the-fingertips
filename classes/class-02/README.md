## What You Will Learn

A deep-dive into OpenAI's ecosystem — from the underlying models to the products built on top of them, to the API economy that lets thousands of developers build with their technology. By the end of this class, you will understand how one company's infrastructure powers a large chunk of the AI products you encounter every day.
