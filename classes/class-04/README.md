## What You Will Learn

Two things in one session: first, a complete map of Google's AI ecosystem — Gemini models, Google AI Studio, NotebookLM, Workspace AI, and Vertex AI. Then a hands-on demonstration of **vibe coding** — using AI to build a real, scroll-animated website purely by describing what you want, without writing a single line of code yourself.

---

## Hands-On Project: Scroll-Animated Website

In the second half of this class, we build a scroll-animated landing page using AI tools only. You will see how to:

1. Describe the site you want in plain English
2. Use an AI coding tool to generate the full HTML/CSS/JS
3. Add scroll animations using prompts — no manual coding
4. Iterate and refine by continuing the conversation

> **You do not need to know how to code to follow along.** That is the point.
